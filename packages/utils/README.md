# @open-insight/utils

Shared Effect-based helpers for the Open Insight workspace.

Currently exports the `Spawn` namespace from [Effect's unstable child-process APIs](https://effect.website), including a typed `SpawnService`, `SpawnError`, and `SpawnExitCodeError`.

## Development

- Install dependencies:

```bash
vp install
```

- Run the unit tests:

```bash
vp test
```

- Build the library:

```bash
vp pack
```
